# LLARS RAG Data

This folder contains versioned, seeded RAG documents that ship with LLARS.

Structure:
- standard/      Seeded documents for the default LLARS assistant
  - general/     Existing standard PDFs
  - llars/       LLARS overview + navigation docs
- legal/         German legal documents for the Legal Assistant
  - bundesgesetze/
  - eu_recht/
- uploads/       Runtime uploads (ignored in git)
- crawls/        Runtime crawler output (ignored in git)
- archive/       Legacy data sets

Note: Only the seeded data in standard/ and legal/ is tracked in git.
